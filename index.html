SketchyOpts#

SketchyOpts is a JAX library for sketching-based methods.

The primary goal is to provide accessible and efficient implementation of these methods to the scientific community. To this end, the library follows design philosophies of JAXopt and provides a familiar interface. We hope the library facilitates research productivity and leads to new ideas and exciting solutions.

Installation#

The latest release of SketchyOpts can be installed using:

pip install sketchyopts
