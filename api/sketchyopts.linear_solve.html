Sketch-and-Solve#

sketchyopts.linear_solve.sketch_and_solve(operator, right_hand_side, solver, sketch_size, *, seed=0, solver_args=None, solver_state=None, sketch_type='gaussian', sketch_args=None)[source]#

Solve a linear system using the sketch-and-solve approach.

The function approximates the solution to the linear system \(Ax = b\) by sketching the operator and right-hand side, and then solving the sketched linear system.

Parameters:

  • operator ‚Äì Linear operator of the system. This is the \(A\) matrix in the system \(Ax = b\).

  • right_hand_side ‚Äì Right-hand side of the system. This is the \(b\) vector in the system \(Ax = b\).

  • solver ‚Äì Solver to use for solving the linear system.

  • sketch_size ‚Äì Size of the sketch.

  • seed ‚Äì Random seed for the sketching matrix (default 0).

  • solver_args ‚Äì Optional arguments for the solver. Keyword-only argument.

  • solver_state ‚Äì Optional state for the solver. This intends to re-use intermediate computation between multiple linear solves with the same operator. Keyword-only argument.

  • sketch_type ‚Äì Type of sketching matrix to use (default gaussian). Options are "gaussian" (Gaussian embedding), "srtt" (subsampled randomized trigonometric transforms), and "sparse-sign" (sparse sign embedding).

  • sketch_args ‚Äì Optional arguments for the sketch method. Keyword-only argument.

Returns:

  • value: Solution to the linear system.

  • result: An integer representing whether the solve was successful or not.

  • stats: Statistics of the solver after solving the linear system.

  • state: State of the solver after solving the linear system.

Return type:

Solution of the linear system that contains the following attributes

Sketch-and-Precondition#

sketchyopts.linear_solve.sketch_and_precondition(operator, right_hand_side, sketch_size, *, seed=0, sketch_type='gaussian', sketch_args=None, tol=1e-06, maxiter=None)[source]#

Solve a linear system using the sketch-and-precondition approach.

The function approximates the solution to the linear system \(Ax = b\) by sketching the operator and use the preconditioned iterative solver to solve the system.

References

  • V. Rokhlin and M. Tygert, A fast randomized algorithm for overdetermined linear least-squares regression. The Proceedings of the National Academy of Sciences, 105(36), 2008.

Parameters:

  • operator ‚Äì Linear operator of the system. This is the \(A\) matrix in the system \(Ax = b\).

  • right_hand_side ‚Äì Right-hand side of the system. This is the \(b\) vector in the system \(Ax = b\).

  • sketch_size ‚Äì Size of the sketch.

  • seed ‚Äì Random seed for the sketching matrix (default 0).

  • sketch_type ‚Äì Type of sketching matrix to use (default gaussian). Options are "gaussian" (Gaussian embedding), "srtt" (subsampled randomized trigonometric transforms), and "sparse-sign" (sparse sign embedding).

  • sketch_args ‚Äì Optional arguments for the sketch method. Keyword-only argument.

  • tol ‚Äì Solution tolerance (default 1e-6).

  • maxiter ‚Äì Maximum number of iterations for the solver (default None). If None, the solver will use \(2\) times the sketch size as its value (same behavior as SciPy LSQR implementation).

Returns:

Approximate solution to the linear least-squares problem and the number of

iterations.
