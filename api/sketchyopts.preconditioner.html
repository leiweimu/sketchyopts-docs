Randomized Nyström Approximation#

sketchyopts.preconditioner.rand_nystrom_approx(A, l, key)[source]#

Randomized Nyström approximation of a positive-semidefinite matrix.

The function computes rank-\(l\) randomized Nyström approximation of a given positive-semidefinite matrix \(A \in \R^{n \times n}\).

It returns the truncated eigen-decomposition \(\hat{A}_{\mathrm{nys}} = U \hat{\Lambda} U^{T}\) where \(U \in \R^{n \times l}\) is orthonormal consisting of eigenvectors as columns and \(\hat{\Lambda} \in \R^{l \times l}\) is diagonal with eigenvalues on its diagonal. Note that output U represents \(U\) whereas S contains only the eigenvalues (not the full diagonal matrix \(\hat{\Lambda}\)).

References

  • H. Li, G. C. Linderman, A. Szlam, K. P. Stanton, Y. Kluger, and M. Tygert, Algorithm 971: An implementation of a randomized algorithm for principal component analysis, ACM Transactions on Mathematical Software (TOMS), 43: 1–14, 2017.

  • J. A. Tropp, A. Yurtsever, M. Udell, and V. Cevher, Fixed-rank approximation of a positive-semidefinite matrix from streaming data, Advances in Neural Information Processing Systems, 30, 2017.

Parameters:

  • A (Union[Array, ndarray, bool, number, bool, int, float, complex, LinearOperator]) – A two-dimensional array representing a positive-semidefinite matrix. This can either be an explicit JAX array or an implicit sketchyopts.util.LinearOperator object.

  • l (int) – Rank of the Nyström approximated matrix \(\hat{A}_{\mathrm{nys}}\) (which coincides with sketch size).

  • key (Union[Array, ndarray, bool, number, bool, int, float, complex]) – A PRNG key used as the random key.

Return type:

tuple[Array, Array]

Returns:

A two-element tuple containing

  • U – Orthonormal matrix \(U\).

  • S – Diagonal entries of \(\hat{\Lambda}\).
