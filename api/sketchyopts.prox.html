Constant Function#

sketchyopts.prox.prox_const(x, hyperparams=None, scaling=1.0)[source]#

Proximal operator for the constant function.

For any constant function \(f(x) \equiv c\) for some constant \(c\), and any positive scaling factor \(\operatorname{scaling}\), the resulting proximal operator is the identity operator

\[\operatorname{prox}_{\operatorname{scaling} \cdot f}(x) = \underset{y}{\argmin} \bigg\{ f(y) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} = x\]
Parameters:

  • x (Any) – Pytree input.

  • hyperparams (Optional[Any]) – Additional parameters of the proximal operator (for function signature consistency). Ignored here.

  • scaling (float) – Scaling factor for the proximal operator (for function signature consistency). Ignored here.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

Norms#

sketchyopts.prox.prox_l1(x, l1reg=None, scaling=1.0)[source]#

Proximal operator for the \(\ell^1\) norm.

The proximal operator of \(f(x) = \lVert x \rVert_1\) for \(x \in \mathbb{R}^d\) is known as the soft-thresholding operator

\[\begin{aligned} \big[ \operatorname{prox} _{\operatorname{scaling} \cdot \operatorname{l1reg} \cdot f}(x) \big]_i &= \underset{y}{\argmin} \bigg\{ \operatorname{l1reg} \, \lVert y \rVert_1 + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x_i \rVert_2^2 \bigg\} \\ &= \begin{cases} ~ 0 & \lvert x_i \rvert \leqslant \operatorname{scaling} \cdot \operatorname{l1reg} \\ ~ x_i - \operatorname{scaling} \cdot \operatorname{l1reg} \cdot \operatorname{sign}(x_i) & \lvert x_i \rvert > \operatorname{scaling} \cdot \operatorname{l1reg} \\ \end{cases} \\ \end{aligned}\]

In lasso regression, the function is used as \(\ell^1\)-regularization.

Parameters:

  • x (Any) – Pytree input.

  • l1reg (Optional[Any]) – Regularization strength. It could either be scalar (same strength across tree leaves) or a pytree with the same structure as x (leaf-wise strength).

  • scaling (float) – Scaling factor for the proximal operator.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_nonnegative_l1(x, l1reg=None, scaling=1.0)[source]#

Proximal operator for the \(\ell^1\) norm on the nonnegative orthant.

The proximal operator of \(f(x) = \lVert x \rVert_1 + \mathbb{1}_{\mathbb{R}_{+}^{d}}(x)\) for \(x \in \mathbb{R}^d\) is given by

\[\begin{aligned} \big[ \operatorname{prox} _{\operatorname{scaling} \cdot \operatorname{l1reg} \cdot f}(x) \big]_i &= \underset{y}{\argmin} \bigg\{ \operatorname{l1reg} \, \big(\lVert y \rVert_1 + \mathbb{1}_{\mathbb{R}_{+}}(y)\big) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x_i \rVert_2^2 \bigg\} \\ &= \underset{y \geqslant 0}{\argmin} \bigg\{ \operatorname{l1reg} \, \lVert y \rVert_1 + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x_i \rVert_2^2 \bigg\} \\ &= \begin{cases} ~ 0 & x_i \leqslant \operatorname{scaling} \cdot \operatorname{l1reg} \\ ~ x_i - \operatorname{scaling} \cdot \operatorname{l1reg} & x_i > \operatorname{scaling} \cdot \operatorname{l1reg} \\ \end{cases} \\ \end{aligned}\]

where \(\mathbb{1}_{\mathbb{R}_{+}^{d}}\) is the indicator function of nonnegative orthant \(\mathbb{R}_{+}^{d} \colonequals \{x \in \mathbb{R}^d \mid x \succcurlyeq 0\}\)

\[\mathbb{1}_{\mathbb{R}_{+}^{d}}(a) = \begin{cases} ~ 0 & a_i \geqslant 0 ~\text{for}~i = 1, \cdots, d \\ ~ \infty & \text{otherwise} \end{cases}\]
Parameters:

  • x (Any) – Pytree input.

  • l1reg (Optional[float]) – Regularization strength. It could either be scalar (same strength across tree leaves) or a pytree with the same structure as x (leaf-wise strength).

  • scaling (float) – Scaling factor for the proximal operator.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_elastic_net(x, hyperparams=None, scaling=1.0)[source]#

Proximal operator for the elastic net.

The proximal operator of

\[f_{\operatorname{hyperparams}}(x) = \operatorname{hyperparams}_0 \, \lVert x \rVert_1 + \frac{\operatorname{hyperparams}_1}{2} \, \lVert x \rVert_2^2\]

for \(x \in \mathbb{R}^d\) is given by soft-thresholding followed by multiplicative shrinkage

\[\begin{aligned} \big[ \operatorname{prox} _{\operatorname{scaling} \cdot f_{\operatorname{hyperparams}}}(x) \big]_i &= \bigg[\underset{y}{\argmin} \bigg\{ f_{\operatorname{hyperparams}}(y) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\}\bigg]_i \\ &= \frac{ \psi_{\operatorname{scaling} \cdot \operatorname{hyperparams}_0}(x_i) }{1 + \operatorname{scaling} \cdot \operatorname{hyperparams}_1} \\ \end{aligned}\]

where \(\psi_{\operatorname{scaling} \cdot \operatorname{hyperparams}_0}\) is the soft-thresholding operator parameterized by \(\operatorname{scaling} \cdot \operatorname{hyperparams}_0\)

\[\psi(x_i) = \begin{cases} ~ 0 & \lvert x_i \rvert \leqslant \operatorname{scaling} \cdot \operatorname{hyperparams}_0 \\ ~ x_i - \operatorname{scaling} \cdot \operatorname{hyperparams}_0 \cdot \operatorname{sign}(x_i) & \lvert x_i \rvert > \operatorname{scaling} \cdot \operatorname{hyperparams}_0 \\ \end{cases}\]
Parameters:

  • x (Any) – Pytree input.

  • hyperparams (Optional[tuple[Any, Any]]) – A tuple, where both hyperparams[0] and hyperparams[1] can be either floats or pytrees with the same structure as x.

  • scaling (float) – Scaling factor for the proximal operator.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_l2(x, l2reg=1.0, scaling=1.0)[source]#

Proximal operator for the \(\ell^2\) norm.

The proximal operator of \(f(x) = \lVert x \rVert_2\) for \(x \in \mathbb{R}^d\) is known as the block soft-thresholding operator

\[\begin{aligned} \operatorname{prox} _{\operatorname{scaling} \cdot \operatorname{l2reg} \cdot f}(x) &= \underset{y}{\argmin} \bigg\{ \operatorname{l2reg} \, \lVert y \rVert_2 + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \begin{cases} ~ 0 & \lVert x \rVert_2 \leqslant \operatorname{scaling} \cdot \operatorname{l2reg} \\ ~ \bigg(1 - \dfrac {\operatorname{scaling} \cdot \operatorname{l2reg}} {\lVert x \rVert_2}\bigg) \, x & \lVert x \rVert_2 > \operatorname{scaling} \cdot \operatorname{l2reg} \\ \end{cases} \\ \end{aligned}\]

In group lasso, the function (also known as \(\ell^1/\ell^2\) norm) takes the form

\[f(x) = \sum_{g \in \mathcal{G}} \lVert x_{\mathcal{I}_g} \rVert_2\]

where \(\mathcal{G}\) is a partition of \([d]\) and \(\mathcal{I}_g\) is the index set belonging to the \(g\)th group. The corresponding proximal operator in this case is given by

\[\big[ \operatorname{prox} _{\operatorname{scaling} \cdot \operatorname{l2reg} \cdot f}(x) \big]_{i} = \begin{cases} ~ 0 & \lVert x_{\mathcal{I}_g} \rVert_2 \leqslant \operatorname{scaling} \cdot \operatorname{l2reg} \\ ~ \bigg(1 - \dfrac {\operatorname{scaling} \cdot \operatorname{l2reg}} {\lVert x_{\mathcal{I}_g} \rVert_2}\bigg) \, x_{i} & \lVert x_{\mathcal{I}_g} \rVert_2 > \operatorname{scaling} \cdot \operatorname{l2reg} \\ \end{cases} \\\]

for all \(i \in \mathcal{I}_g\) and \(g \in \mathcal{G}\).

Note

The proximal operator is implemented for the \(\ell^2\) norm (i.e. single group case). To use it for group lasso, one can use jax.vmap() or jax.tree.map() to apply the proximal mapping to groups as shown in the following example.

Example

import jax
from sketchyopts.prox import prox_l2

# array with 2 or more dimensions and first axis across groups
def prox_group_lasso_array(x, l2reg=1.0, scaling=1.0):
  # collapse each group into a single dimension
  groups = jax.lax.collapse(x, 1)
  # apply the operator to each group
  result = jax.vmap(prox_l2, (0, None, None))(groups, l2reg, scaling)
  # make result the original shape
  return result.reshape(x.shape)

# pytree with groups as leaves
def prox_group_lasso_pytree(x, l2reg=1.0, scaling=1.0):
  # define partial function that only depends on the pytree
  partial_fun = lambda u: prox_l2(u, l2reg, scaling)
  # apply the operator to each group
  result = jax.tree.map(partial_fun, x)
  return result
Parameters:

  • x (Any) – Pytree input.

  • l2reg (Optional[float]) – Regularization strength.

  • scaling – Scaling factor for the proximal operator.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_l2_squared(x, l2reg=1.0, scaling=1.0)[source]#

Proximal operator for the squared \(\ell^2\) norm.

The proximal operator of \(f(x) = (1/2) \, \lVert x \rVert_2^2\) for \(x \in \mathbb{R}^d\) is given by

\[\begin{aligned} \operatorname{prox} _{\operatorname{scaling} \cdot \operatorname{l2reg} \cdot f}(x) &= \underset{y}{\argmin} \bigg\{ \frac{\operatorname{l2reg}}{2} \, \lVert y \rVert_2^2 + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \frac{1}{1 + \operatorname{scaling} \cdot \operatorname{l2reg}} \, x \\ \end{aligned}\]

In ridge regression, the function is used as \(\ell^2\)-regularization.

Parameters:

  • x (Any) – Pytree input.

  • l2reg (Optional[float]) – Regularization strength.

  • scaling – Scaling factor for the proximal operator.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_nonnegative_l2_squared(x, l2reg=1.0, scaling=1.0)[source]#

Proximal operator for the squared \(\ell^2\) norm on the nonnegative orthant.

The proximal operator of \(f(x) = (1/2) \, \lVert x \rVert_2^2 + \mathbb{1}_{\mathbb{R}_{+}^{d}}(x)\) for \(x \in \mathbb{R}^d\) is given by

\[\begin{aligned} \operatorname{prox} _{\operatorname{scaling} \cdot \operatorname{l2reg} \cdot f}(x) &= \underset{y}{\argmin} \bigg\{ \operatorname{l2reg} \, \bigg( \frac{1}{2} \, \lVert y \rVert_2^2 + \mathbb{1}_{\mathbb{R}_{+}^{d}}(y) \bigg) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \underset{y \succcurlyeq 0}{\argmin} \bigg\{ \frac{\operatorname{l2reg}}{2} \, \lVert y \rVert_2^2 + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \bigg( \frac{1}{1 + \operatorname{scaling} \cdot \operatorname{l2reg}} \, x \bigg)_{+} \\ \end{aligned}\]

where \(\mathbb{1}_{\mathbb{R}_{+}^{d}}\) is the indicator function of nonnegative orthant \(\mathbb{R}_{+}^{d} \colonequals \{x \in \mathbb{R}^d \mid x \succcurlyeq 0\}\)

\[\mathbb{1}_{\mathbb{R}_{+}^{d}}(a) = \begin{cases} ~ 0 & a_i \geqslant 0 ~\text{for}~i = 1, \cdots, d \\ ~ \infty & \text{otherwise} \end{cases}\]

and \((a)_{+} \colonequals \big[\max \{a_i, 0\}\big]_{i = 1, \cdots, d}\) takes element-wise positive part of \(a\).

Parameters:

  • x (Any) – Pytree input.

  • l2reg (Optional[float]) – Regularization strength.

  • scaling (float) – Scaling factor for the proximal operator.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

Indicators (projections)#

sketchyopts.prox.prox_nonnegative(x, hyperparams=None, scaling=1.0)[source]#

Proximal operator for indicator of the nonnegative orthant.

The proximal operator of indicator \(f(x) = \mathbb{1}_{\mathbb{R}_{+}^{d}}(x)\) for \(x \in \mathbb{R}^d\) is the projection onto the nonnegative orthant. Here \(\mathbb{1}_{\mathbb{R}_{+}^{d}}\) is the indicator function of nonnegative orthant \(\mathbb{R}_{+}^{d} \colonequals \{x \in \mathbb{R}^d \mid x \succcurlyeq 0\}\)

\[\mathbb{1}_{\mathbb{R}_{+}^{d}}(a) = \begin{cases} ~ 0 & a_i \geqslant 0 ~\text{for}~i = 1, \cdots, d \\ ~ \infty & \text{otherwise} \end{cases}\]

The proximal operator is given by

\[\begin{aligned} \operatorname{prox}_{\operatorname{scaling} \cdot f}(x) &= \underset{y}{\argmin} \bigg\{ \mathbb{1}_{\mathbb{R}_{+}}(y) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \underset{y \succcurlyeq 0}{\argmin} \bigg\{ \lVert y - x \rVert_2^2 \bigg\} \\ &= (x)_{+} \\ \end{aligned}\]

where \((a)_{+} \colonequals \big[\max \{a_i, 0\}\big]_{i = 1, \cdots, d}\) takes element-wise positive part of \(a\).

Parameters:

  • x (Any) – Pytree input.

  • hyperparams (Optional[Any]) – Additional parameters of the proximal operator (for function signature consistency). Ignored here.

  • scaling (float) – Scaling factor for the proximal operator (for function signature consistency). Ignored here.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_box(x, hyperparams, scaling=1.0)[source]#

Proximal operator for indicator of a box (high-dimensional closed interval).

The proximal operator of indicator \(f(x) = \mathbb{1}_{\mathcal{C}}(x)\) for set

\[\mathcal{C} \colonequals \{x \in \mathbb{R}^d \mid \operatorname{hyperparams}_0 \preccurlyeq x \preccurlyeq \operatorname{hyperparams}_1 \}\]

is the projection onto the set \(\mathcal{C}\). Here \(\mathbb{1}_ {\mathcal{C}}\) is the indicator function of the box \(\mathcal{C}\)

\[\mathbb{1}_{\mathcal{C}}(a) = \begin{cases} ~ 0 & \big[\operatorname{hyperparams}_0\big]_i \leqslant a_i \leqslant \big[\operatorname{hyperparams}_1\big]_i ~\text{for}~i = 1, \cdots, d \\ ~ \infty & \text{otherwise} \end{cases}\]

The proximal operator is given by

\[\begin{aligned} \operatorname{prox}_{\operatorname{scaling} \cdot f}(x) &= \underset{y}{\argmin} \bigg\{ \mathbb{1}_{\mathcal{C}}(y) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \underset{y \in \mathcal{C}}{\argmin} \bigg\{ \lVert y - x \rVert_2^2 \bigg\} \\ &= \max\big\{ \operatorname{hyperparams}_0, \, \min\{x, \, \operatorname{hyperparams}_1\} \big\} \\ \end{aligned}\]

where both the minimum and maximum are element-wise operations. Intuitively, the proximal operator clamps \(x\) into the interval.

Parameters:

  • x (Any) – Pytree input.

  • hyperparams (tuple) – A tuple (lower, upper) specifying lower and upper bounds of the box. Both bounds can be either scalar values or pytrees of the same structure as x.

  • scaling (float) – Scaling factor for the proximal operator (for function signature consistency). Ignored here.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_hyperplane(x, hyperparams, scaling=1.0)[source]#

Proximal operator for indicator of a hyperplane.

The proximal operator of indicator \(f(x) = \mathbb{1}_{\mathcal{H}}(x)\) for set

\[\mathcal{H} \colonequals \{x \in \mathbb{R}^d \mid \operatorname{hyperparams}_0^{\mathsf{T}} x = \operatorname{hyperparams}_1 \}\]

is the projection onto the set \(\mathcal{H}\). Here \(\mathbb{1}_ {\mathcal{H}}\) is the indicator function of the hyperplane \(\mathcal{H}\)

\[\mathbb{1}_{\mathcal{H}}(a) = \begin{cases} ~ 0 & \operatorname{hyperparams}_0^{\mathsf{T}} a = \operatorname{hyperparams}_1 \\ ~ \infty & \text{otherwise} \end{cases}\]

The proximal operator is given by

\[\begin{aligned} \operatorname{prox}_{\operatorname{scaling} \cdot f}(x) &= \underset{y}{\argmin} \bigg\{ \mathbb{1}_{\mathcal{H}}(y) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \underset{y \in \mathcal{H}}{\argmin} \bigg\{ \lVert y - x \rVert_2^2 \bigg\} \\ &= x - \frac { \operatorname{hyperparams}_0^{\mathsf{T}} x - \operatorname{hyperparams}_1 }{ \lVert \operatorname{hyperparams}_0 \rVert_2^2 } \, \operatorname{hyperparams}_0 \\ \end{aligned}\]
Parameters:

  • x (Any) – Pytree input.

  • hyperparams (tuple) – A tuple (a, b) specifying both the normal vector and the offset of the hyperplane. The normal vector a is a pytree with the same structure as x and the offset b is a scalar.

  • scaling (float) – Scaling factor for the proximal operator (for function signature consistency). Ignored here.

Return type:

Any

Returns:

Output pytree, with the same structure as x.

sketchyopts.prox.prox_halfspace(x, hyperparams, scaling=1.0)[source]#

Proximal operator for indicator of a halfspace.

The proximal operator of indicator \(f(x) = \mathbb{1}_{\mathcal{H}}(x)\) for set

\[\mathcal{H} \colonequals \{x \in \mathbb{R}^d \mid \operatorname{hyperparams}_0^{\mathsf{T}} x \leqslant \operatorname{hyperparams}_1 \}\]

is the projection onto the set \(\mathcal{H}\). Here \(\mathbb{1}_ {\mathcal{H}}\) is the indicator function of the halfspace \(\mathcal{H}\)

\[\mathbb{1}_{\mathcal{H}}(a) = \begin{cases} ~ 0 & \operatorname{hyperparams}_0^{\mathsf{T}} a \leqslant \operatorname{hyperparams}_1 \\ ~ \infty & \text{otherwise} \end{cases}\]

The proximal operator is given by

\[\begin{aligned} \operatorname{prox}_{\operatorname{scaling} \cdot f}(x) &= \underset{y}{\argmin} \bigg\{ \mathbb{1}_{\mathcal{H}}(y) + \frac{1}{2 \cdot \operatorname{scaling}} \, \lVert y - x \rVert_2^2 \bigg\} \\ &= \underset{y \in \mathcal{H}}{\argmin} \bigg\{ \lVert y - x \rVert_2^2 \bigg\} \\ &= x - \frac { (\operatorname{hyperparams}_0^{\mathsf{T}} x - \operatorname{hyperparams}_1)_{+} }{ \lVert \operatorname{hyperparams}_0 \rVert_2^2 } \, \operatorname{hyperparams}_0 \\ \end{aligned}\]

where \((a)_{+} \colonequals \max \{a, 0\}\) takes the positive part of \(a\).

Parameters:

  • x (Any) – Pytree input.

  • hyperparams (tuple) – A tuple (a, b) specifying both the normal vector and the offset of the hyperplane that defines the halfspace. The normal vector a is a pytree with the same structure as x and the offset b is a scalar.

  • scaling (float) – Scaling factor for the proximal operator (for function signature consistency). Ignored here.

Return type:

Any

Returns:

Output pytree, with the same structure as x.
