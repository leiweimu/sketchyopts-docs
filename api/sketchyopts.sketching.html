Randomized Sketching Class#

class sketchyopts.sketching.RandomizedSketching(operator, sketch_size, key, tags=())[source]#

Abstract class for randomized sketching.

The class defines the interface for a randomized sketching operator. The class represents a sketched version of a linear operator.

abstract apply_sketch(vector)[source]#

Apply the sketching to a vector.

The method applies the sketching direcrly to a vector of the compatible size (i.e. the size of the operator’s output). This differs from the matrix-vector product, which applies the sketched operator to a vector of the size of the operator’s input.

Mathematically if we denote the operator as \(A\) and the sketching matrix as \(S\), the method computes \(S x\) whereas the matrix-vector product computes \(S A x\).

Parameters:

vector – Vector that the sketching applies to. The vector should be a pytree of floating-point arrays whose structure matches the output structure of the operator.

Return type:

Array

Returns:

Sketched vector. The output is a floating-point array of length equal to the sketch size.

abstract as_matrix()[source]#

Materialize the sketched operator.

The method materializes the sketched operator as a matrix. Since many linear operators are defined implicitly and sketching is not necessarily stored as a full matrix, materializing can be a computationally expensive operation.

Return type:

Array

Returns:

Sketched operator as a 2D JAX array. The output is a floating-point array of shape \((\text{{sketch\_size}}, \text{{operator\_input\_size}})\).

abstract generate_sketching()[source]#

Generate the sketching.

The method generates the sketching internally. The method is called when the sketching is not provided during the initialization. Users can also call the method to re-generate the sketching.

Return type:

None

Sketching Types#

class sketchyopts.sketching.GaussianEmbedding(operator, sketch_size, key, *, tags=(), sketch_matrix=None)[source]#

Gaussian embedding sketching operator.

The class defines a Gaussian embedding sketching operator. The sketching is defined by a Gaussian random matrix (each entry is independently drawn from a normal distribution with zero mean and variance \(1 / \text{sketch\_size}\)).

Parameters:

  • operator – Linear operator to sketch.

  • sketch_size – Size of the sketch.

  • key – Random key.

  • tags – Tags for the sketched operator (default None).

  • sketch_matrix – Sketching matrix (default None). If provided, the operator uses the provided matrix for sketching. Otherwise, the operator generates the sketching matrix at initialization.

class sketchyopts.sketching.SRTT(operator, sketch_size, key, *, tags=(), permutation=None, sign_flip=None, restriction=None)[source]#

Subsampled randomized trigonometric transforms (SRTT) sketching operator.

The class defines a subsampled randomized trigonometric transforms (SRTT) sketching operator. The sketching \(S\) is defined by a random restriction \(R\), a discrete cosine transform \(F\), a random sign flip \(E\), and a random permutation \(\Gamma\):

\[S = \sqrt{\frac{n}{d}} E F R \Gamma \]

where \(n\) is the size of the operator’s output, \(d\) is the size of the sketch.

If the permutation, sign flip, and restriction arrays are provided, the operator uses the provided arrays for sketching. Otherwise, the operator generates the matrices at initialization.

Parameters:

  • operator – Linear operator to sketch.

  • sketch_size – Size of the sketch.

  • key – Random key.

  • tags – Tags for the sketched operator (default None).

  • permutation – Permutation array (default None). It is stored as a vector of permutation of the operator’s output indices.

  • sign_flip – Sign flip array (default None). It is stored as a vector of random signs independently drawn uniform \(\{\pm 1\}\)) of size of the operator’s output.

  • restriction – Restriction array (default None). It is stored as a vector of randomly drawn indices of size of the sketch from the operator’s output indices.

class sketchyopts.sketching.SparseSignEmbedding(operator, sketch_size, key, *, tags=(), sparsity=None, coord_matrix=None, sign_matrix=None)[source]#

Sparse sign embedding sketching operator.

The class defines a sparse sign embedding sketching operator. The sketching \(S\) is defined by a sparse matrix where each column has exactly \(\zeta\) non-zero entries. Each non-zero entry is independently drawn uniformly from \(\{\pm 1\}\), and its location is randomly chosen from \(\{1, \cdots, d\}\) where \(d\) is the size of the sketch. Mathematically, the sketching takes the form:

\[S = \sqrt{\frac{1}{\xi}} \big[s_1, \cdots, s_n\big] \]

Here \(n\) is the size of the operator’s output.

If the non-zero indices and corresponding random signs are both provided, the operator uses the provided arrays for sketching. Otherwise, the operator generates the matrices at initialization.

The sparsity parameter \(\zeta\) takes the recommended practical value of \(\operatorname{min}\{8, d\}\) [1].

References

[1]

J. A. Tropp, A. Yurtsever, M. Udell, and V. Cevher, Streaming Low-Rank Matrix Approximation with an Application to Scientific Simulation, SIAM Journal on Scientific Computing, 41(4), 2019.

Parameters:

  • operator – Linear operator to sketch.

  • sketch_size – Size of the sketch.

  • key – Random key.

  • tags – Tags for the sketched operator (default None).

  • sparsity – Sparsity parameter \(\zeta\) (default None). If provided, the operator uses the provided value for the sparsity. Otherwise, the operator uses the recommended value of \(\operatorname{min}\{8, d\}\).

  • coord_matrix – Coordinate matrix (default None).

  • sign_matrix – Sign matrix (default None).
